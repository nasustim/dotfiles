# Layout

- `confs/.vimrc` - Vim configuration
- `confs/.zshrc`, `confs/.zshenv`, `confs/.zsh/` - Zsh config + git prompt/completion
- `confs/.config/tmux/` - Tmux configuration (XDG)
- `confs/.config/git/` - Git config, ignore patterns, multiple identities
- `confs/.config/mise/` - Mise version manager (replaces asdf)
- `confs/.config/gh/`, `confs/.config/gwq/` - gh CLI and gwq config
- `confs/.claude/` - Claude Code settings, CLAUDE.md, agent-memory
- `confs/.codex/`, `confs/.copilot/` - AGENTS.md / copilot-instructions.md symlinked to CLAUDE.md
- `.mise.toml` (repo root) - symlinked into `confs/.config/mise/config.toml` by link.sh
- `lib/` - link.sh / unlink.sh scripts
